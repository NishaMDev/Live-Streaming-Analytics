Sentiment Analysis Model

OPENAI's GPT-3 is a state-of-the-art natural language processing (NLP) model. OpenAI Completion is an advanced natural language processing (NLP) tool that can generate text based on a given prompt. For sentiment analysis, we used OpenAI Completion to generate text based on a given prompt, which was a sentence on the probability of sentiment. We set the model to "text-davinci-003," the temperature to 0, the max_tokens to 1000, the top_p to 1.0, the frequency_penalty to 0.0, and the presence_penalty to 0.0.


Using these settings, OpenAI Completion was able to generate text based on the given prompt. This text was then used as input to a sentiment analysis model to detect the sentiment of the generated text. This enabled us to gain a better understanding of the sentiment of the given prompt and to improve the accuracy of our sentiment analysis model.

Chat Summarization

The chat which is stored in the database will be fed to the pretrained GPT-3 DaVinci model which will provide the abstractive summary of the chat. This model is based on transformer architecture. The model takes prompt messages along with chat messages as the input and then generates output sentences from them.. This summary will help the subscriber or any user to get an understanding of the ongoing discussion. The UI will show the generated summary once the user clicks on the Chat Summary button.
